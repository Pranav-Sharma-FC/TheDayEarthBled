## Game Description
In this game you fight off enemies for a certain amount of waves untill you get to 5000 points or die. Assets unfinished in its current state.

This project is local multiplayer, and will work best with four controllers, however, for simplicities sake, here are the keyboard keys
## Controls
WASD - Move  
TFGH - Move Port (Left) Aiming, Shoot with 1  
IKJL - Move Stern (Back) Aiming, Shoot with 2  
Arrow Keys - Move Starboard (Right) Aiming, Shoot with 3  
Initial Screen - Press 0 to continue   
Press F4 to Pause
