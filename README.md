This project will continue to be worked on for a little bit beyond the moonshot deadline.
This branch will not get updated past Friday, to avoid any confusion. All the work I have done up till this point has been saved here. The total if you look at this should be around 9 hours.
This project is local multiplayer, and will work best with four controllers, however, for simplicities sake, here are the keyboard keys
WASD - Move
TFGH - Move Port (Left) Aiming, Shoot with 1
IKJL - Move Stern (Back) Aiming, Shoot with 2
Arrow Keys - Move Starboard (Right) Aiming, Shoot with 3
Initial Screen - Press 0 to continue 
Press F4 to Pause
