# The Day Earth Bled
A Space shooter requiring four players.
Defend earth from an alien invasion, and allow humanity to escape.

### RFAH
We had:
Sci-fi - Present throughout the game
Redbud - Present throughout the game
Inventory - Mainly in GameManager Script, with connections to player, enemy, and UI panel scripts
Dialouge - Mainly in Gamemanager and UI Panel Script

### OOP
Entity Script houses many abstract classes
Private methods and Variables: Player and Enemy
Is operator: Player, Enemy and Bullet
Generics: Player and Enemy
Enum: Gamemanager

### Godot Features
Custom signals are present in the UI Panel (Reciever) and Gamemanager (Broadcaster)
PackedScenes are used for Bullet instantiation in script
_Ready is used in almost every single script (Gamemanager for quick check)
Custom inputs are available in project settings
Main Menu Changes scenese in the MainMenu Script
